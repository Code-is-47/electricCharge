# electricCharge

宁波诺丁汉大学生活区查询寝室电费

## 使用方法

微信公众号关注“Yiming的小站”，按照譬如“19-401”的格式发送房间号

暂时只支持15，19，23幢宿舍楼

也可以自行补充密钥后使用
