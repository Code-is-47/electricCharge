# electricCharge

宁波诺丁汉大学生活区查询寝室电费

## 说明

由于没有常规渠道获取Cookie，只能定期抓包更新到数据库中，周期未知

如果Cookie过期，电费数据能够正常获取，但是付款链接点击后会显示下单账户和付款账户不匹配

## 使用方法

微信公众号关注“Yiming的小站”，按照譬如“19-401”的格式发送房间号

暂时只支持15，19，23幢宿舍楼
